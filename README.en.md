# HCSDKtoRTMP

#### Description
因海康nvr使用rtsp转封装为rtmp流会存在带宽限制，无法满足项目需求。故而通过海康sdk二次开发捕获码流数据，将捕获到的ps流转封装为rtmp推到nginx服务中。
